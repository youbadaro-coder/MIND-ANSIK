# 🚀 Shorts Factory Pro (Nova Engine)

회원가입 없이 로컬에서 바로 사용하는 프리미엄 쇼츠 생성기입니다. 다운로드받으신 `ShortsMagician`을 기반으로 더욱 강력한 기능과 세련된 디자인으로 업그레이드되었습니다.

## ✨ 주요 기능
- **회원가입 불필요**: 로컬 환경에서 개인적으로 즉시 실행 가능합니다.
- **Nova AI 엔진**: Gemini 2.0 Flash를 사용하여 더 자극적이고 몰입감 있는 대본을 생성합니다.
- **유리잔 디자인 (Glassmorphism)**: 고급스러운 대시보드 UI를 통해 작업 진행 상황을 실시간으로 모니터링합니다.
- **자동화 파이프라인**: 대본 작성 -> 영상 소스 수집 -> 나레이션(TTS) -> 배경음악 믹싱 -> 최종 렌더링이 한 번에 이루어집니다.

## 🛠 실행 방법
1. **API 키 설정**: `backend/.env.example` 파일을 `.env`로 이름을 바꾼 후, `GEMINI_API_KEY`와 `PEXELS_API_KEY`를 입력하세요.
2. **원클릭 실행**: 루트 폴더에 있는 `run_nova.bat` 파일을 더블 클릭합니다.
3. **대시보드 접속**: 브라우저에서 자동으로 열리는 대시보드에서 주제를 입력하고 생성 버튼을 누르세요.

## 📂 폴더 구조
- `backend/`: Flask 서버 및 비디오 제작 엔진
- `frontend/`: 대시보드 UI (index.html)
- `data/`: 작업 히스토리 저장
- `output/`: 생성된 최종 영상 파일 저장
